# Peregrine Digital Sales Room

A Digital Sales Room (DSR) built for Peregrine Hotel Group by [Reeco](https://reeco.com) — a personalized, shareable microsite for pre-sales and post-meeting follow-up.

## Versions

| File | Description |
|------|-------------|
| [`index.html`](./index.html) | **v4 — Primary version** (latest, recommended) |
| [`standalone.html`](./standalone.html) | Fully self-contained version |
| [`v3.html`](./v3.html) | Version 3 |
| [`v3-print.html`](./v3-print.html) | Version 3 — print-optimized layout |
| [`v2.html`](./v2.html) | Version 2 |
| [`v1.html`](./v1.html) | Original version |

## How to View

**Online (GitHub Pages):**  
Open the live URL: `https://zanders-commits.github.io/peregrine-digital-sales-room/`

**Locally:**  
Clone the repo and open `index.html` in any modern browser — no build step required.

```bash
git clone https://github.com/zanders-commits/peregrine-digital-sales-room.git
cd peregrine-digital-sales-room
open index.html   # macOS
# or
start index.html  # Windows
```
